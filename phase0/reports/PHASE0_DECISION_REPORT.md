## 三、失败分析

### t2_s3（近景特写）

**失败原因**: 阿里云 Green Net 内容安全审核拦截  
**Prompt**: 近景头肩镜头，Lena 面部特写 + 浅景深模糊背景  
**分析**: 面部特写触发了阿里内容安全的人脸/肖像审核规则  
**对策**: 近景特写 prompt 增加"portrait photography style, editorial fashion"等安全锚点词

### 网络闪断

测试过程中多次遇到 `ConnectionResetError(54, 'Connection reset by peer')`。重试即可恢复，非 API 服务问题，但生产环境需在 SDK 中内置自动重试（3 次 + 指数退避）。

## 四、API 运行特征

- **端点**: `https://dashscope.aliyuncs.com/api/v1/services/aigc/video-generation/video-synthesis`
- **模型名**: `happyhorse-1.0-t2v` / `happyhorse-1.1-t2v`
- **r2v 端点**: 同 t2v，`input.media` 数组传入参考图 URL 或 base64
- **异步模式**: 提交 → task_id → 轮询 → 获取视频 URL
- **生成速度**: 10-210s，中位数 ~170s（1080p 5 秒视频）
- **输出格式**: 1080p, 16:9, H.264, ~9.5Mbps；文件名含 `_refiner_watermark.mp4`
- **水印**: 所有输出含阿里云水印，API 无去水印参数 — 此为 Phase 1 需解决的核心问题
- **并发限制**: 测试 Key 2 条/批，批次间隔 60s 无触发限流
- **OSS 签名 URL**: 有时效性，需及时下载
